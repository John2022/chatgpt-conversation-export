Permissions and privacy

The extension runs only on chatgpt.com. It reads the currently open ChatGPT conversation through ChatGPT’s authenticated backend API and exports it locally. It does not send conversations, files, metadata, or tokens to any third-party server. The session token is used only in memory during export and is never written to exported files.
