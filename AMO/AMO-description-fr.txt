ChatGPT Conversation Export permet d’exporter localement la conversation ChatGPT actuellement ouverte dans Firefox.

L’extension est conçue pour les utilisateurs qui travaillent avec des conversations ChatGPT longues ou complexes et qui ont besoin d’un moyen fiable de conserver, relire ou reprendre leur travail. Les très longues conversations peuvent devenir lentes, difficiles à parcourir, ou plus difficiles à continuer correctement. L’export permet de conserver une sauvegarde locale, d’extraire le contexte utile et de redémarrer depuis une conversation plus propre lorsque c’est nécessaire.

L’extension propose deux modes principaux :

- Conversation lisible : une version claire de l’échange, adaptée à la relecture, au classement, à l’impression ou à la préparation d’un prompt de reprise.
- Archive complète : un export plus détaillé avec davantage de métadonnées, utile pour la vérification, le diagnostic ou la conservation à long terme.

Formats disponibles :

- JSON lisible ;
- HTML lisible ;
- JSON complet ;
- HTML complet.

L’export HTML affiche les éléments courants d’une conversation : titres, listes, tableaux, blocs de code, code inline, citations, références de sources, pièces jointes et galeries d’images lorsque les métadonnées sont disponibles.

L’extension peut être utilisée depuis la popup de Firefox ou depuis un bouton d’export intégré dans la page ChatGPT.

Modèle de confidentialité :

- les exports sont générés localement dans le navigateur ;
- l’extension n’exploite aucun serveur tiers ;
- les conversations, paramètres et tokens de session ne sont pas envoyés vers un service externe ;
- la session ChatGPT déjà active dans l’onglet est utilisée uniquement pour récupérer la conversation courante pendant l’export.

Limites :

ChatGPT peut modifier son interface ou ses API internes. Dans ce cas, une mise à jour de l’extension peut être nécessaire.

Certaines images, sources ou pièces jointes peuvent dépendre des métadonnées effectivement retournées par ChatGPT. Lorsque le contenu direct n’est pas disponible, l’extension conserve les références utiles au lieu de générer un faux aperçu.

ChatGPT Conversation Export est une extension non officielle. Elle n’est pas développée, approuvée ni affiliée à OpenAI.
