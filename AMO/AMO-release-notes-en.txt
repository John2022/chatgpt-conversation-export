Release candidate `v1.0.0-rc5` of ChatGPT Conversation Export.

Main features:

- export the open ChatGPT conversation;
- JSON and HTML formats;
- readable conversation mode;
- complete archive mode;
- integrated button inside the ChatGPT interface;
- Firefox toolbar popup;
- settings page;
- HTML rendering for headings, lists, tables, code blocks, inline code, quotes, web sources and image galleries;
- local operation without a third-party server.

Changes in this release candidate:

- refreshed the extension icon set with PNG icons in multiple sizes;
- keeps the Firefox `data_collection_permissions` manifest declaration with `none`;
- keeps the DOM-construction cleanup that avoids AMO `innerHTML` validation warnings;
- keeps the technical diagnostic export limited to API and parser summaries, without writing the full conversation content;
- keeps the clearer localized diagnostic label and hidden-by-default diagnostic action;
- keeps the image carousel rendering fix introduced in rc3.

Direct PDF export is not included yet. Exported HTML files can be printed to PDF from Firefox.
